📑 Linux安装Matlab流程
Posted on Tue, Oct 2, 2018 Matlab

一般安装文件涉及到三个（假设在~/Matlab/中）：

***_Crack.zip
***dvd1.iso
***dvd2.iso

先解压破解补丁：

unzip ***_Crack.zip [-d hello]

[]中的内容可选。

然后创建目录，并挂载：

mkdir ~/iso
sudo mount -o loop ~/Matlab/***dvd1.iso
cd ~
sudo ~/iso/install

输入破解补丁解压后的readme.txt中的key。 安装到一半，或者80%等各种进度，提示挂载dvd2.iso：

sudo mount -o loop ~/Matlab/***dvd2.iso

完成后，将破解补丁解压的目录中破解文件移动到Matlab安装目录的/bin/glnx64/中对应，然后将license\_standalone.lic移动到Matlab\_install_dir/licenses/下。
