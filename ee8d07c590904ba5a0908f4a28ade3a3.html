📑 viPlugin破解方法
Posted on Wed, May 15, 2019 Crack vim
  1. http://viplugin.com/此处下载viPlugin包viPlugin_2.14.1.zip。
  2. 在eclipse中安装viPlugin，地址填写http://viplugin.com。
  3. 解压viPlugin_2.14.1.zip得到com.mbartl.viplugin.eclipse.layer_2.14.1.jar。
  4. 再解压com.mbartl.viplugin.eclipse.layer_2.14.1.jar得到***\\lib\\文件夹下有3个jar包:
commons-codec-1.6.jar
commons-lang-2.1.jar
licensechecker.jar

5.导入这3个包，并运行如下代码：

import com.mbartl.viimplementation.license.CheckLicenseFile;

public class hello{
    public static void main(String[] args) {
        try {
            String lis = CheckLicenseFile
                    .encrypt("xxxx", "xxxx@gmail.com"); //用户和邮箱自己给定
            System.out.println(lis);

        } catch (Exception e) {
            System.out.println(e.getStackTrace());
        } finally {
            System.out.println(";asdfasdf");
        }
    }
}

6.将输出的字符写进eclipse的workspace文件夹下的viPlugin2.lic中
