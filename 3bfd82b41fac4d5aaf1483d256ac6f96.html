📑 Linux安装字体
Posted on Mon, Dec 24, 2018 Linux

字体下载

YaHeiConsolas.tar：https://pan.baidu.com/s/1yid9NVomqtxvH73ZVgnpSg

字体安装

tar -zxvf YaHeiConsolas.tar.gz  //解压压缩包
sudo mkdir -p /usr/share/fonts/vista  //在系统目录下创建自定义字体目录
sudo cp YaHeiConsolas.ttf /usr/share/fonts/vista/  //复制解压出来的字体到刚才创建的目录
sudo chmod 644 /usr/share/fonts/vista/*.ttf  //修改字体权限
cd /usr/share/fonts/vista/  //进入字体目录
sudo mkfontscale && sudo mkfontdir && sudo fc-cache -fv  //刷新并安装字体
